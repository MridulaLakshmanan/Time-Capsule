# 🌌 Time-Capsule-Journal

### *A Simple Web-Based Memory Vault*

> A lightweight front-end web application that allows users to capture, store, and revisit their personal memories through a structured interface.

---

##  Overview  

**Time Capsule** is a browser-based application built using **HTML, CSS, and JavaScript** that enables users to store meaningful moments digitally.

The project focuses on:
- Clean UI design  
- Multi-page navigation flow  
- User interaction using JavaScript  
- Structured memory entry and visualization

## ✨ Features  

📝 **Memory Entry System**  
- Users can input and store personal memories  
- Structured form-based input  

 **Dashboard View**  
- Displays stored entries in an organized way  
- Central place to revisit memories  

🔐 **User Flow Pages**  
- Signup page (`signup.html`)  
- Entry page (`entry.html`)  
- Dashboard (`dashboard.html`)  

 **Responsive UI Design**  
- Styled using custom CSS  
- Clean and minimal layout  

⚙️ **Client-Side Logic**  
- JavaScript handles interactions and data flow (`cap.js`)  

---

## Tech Stack  

- **HTML5** → Structure of web pages  
- **CSS3** → Styling and layout  
- **JavaScript** → Functionality and logic

- User lands on index page
- Signs up via signup page
- Adds memories using entry page
